# ShadowLend SDK

This is the TypeScript SDK for the ShadowLend protocol on Solana, utilizing Arcium for confidental computing.

## Features

- **Confidential Deposits & Withdrawals**: Encrypted interactions using Arcium.
- **Branded Types**: Type-safe wrappers for common data types (U64, PublicKeys, etc.).
- **Functional API**: Tree-shakable instruction builders.
- **Admin Utilities**: Tools for initializing pools and computation definitions.

## Installation

```bash
npm install @shadowlend/sdk
```

## Documentation

Run `npm run docs` to generate the API documentation.
